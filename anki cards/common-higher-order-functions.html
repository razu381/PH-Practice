Most Common Higher Order Functions in JavaScript

JavaScript provides several built-in higher order functions for working with arrays. These functions take other functions as arguments and are very useful for data transformation and iteration.

1. map() – Transform Each Element

Creates a new array by applying a function to every element of the original array.


const numbers = [1, 2, 3];
const squares = numbers.map(n => n * n);
console.log(squares); // [1, 4, 9]

2. filter() – Select Elements

Creates a new array with only the elements that pass a test function.


const numbers = [1, 2, 3, 4];
const even = numbers.filter(n => n % 2 === 0);
console.log(even); // [2, 4]

3. reduce() – Accumulate a Value

Reduces an array to a single value by applying a function to each element and an accumulator.


const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, n) => acc + n, 0);
console.log(sum); // 10

4. forEach() – Run a Function for Each Element

Executes a function for each array element (does not return a new array).


const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach(fruit => {
  console.log(fruit);
});
// Output:
// apple
// banana
// cherry

5. find() – Find the First Match

Returns the first element that matches a condition, or undefined if none found.


const users = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 }
];
const user = users.find(u => u.age > 26);
console.log(user); // { name: 'Bob', age: 30 }
