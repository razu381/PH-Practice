Higher Order Functions in JavaScript

A higher order function is a function that does at least one of the following:

  • Takes one or more functions as arguments
  • Returns a function as its result

They are a key concept in JavaScript and functional programming.

Example 1: Passing a Function as an Argument

Here, map is a higher order function because it takes another function as an argument. 


const numbers = [1, 2, 3];
const doubled = numbers.map(function(n) {
  return n * 2;
});
console.log(doubled); // [2, 4, 6]

Example 2: Returning a Function

This function returns another function, making it higher order.


function greet(message) {
  return function(name) {
    return message + ', ' + name + '!';
  };
}
const sayHello = greet('Hello');
console.log(sayHello('Alice')); // Hello, Alice!

Example 3: Custom Higher Order Function

This function takes another function and applies it to each element of an array. 


function applyToEach(arr, fn) {
  for (let i = 0; i < arr.length; i++) {
    arr[i] = fn(arr[i]);
  }
  return arr;
}
console.log(applyToEach([1, 2, 3], x => x * 3)); // [3, 6, 9]

Example 4: Using forEach (Built-in Higher Order Function)

forEach takes a function and runs it for every element in the array. 


[1, 2, 3].forEach(function(item) {
  console.log(item);
});
// Output:
// 1
// 2
// 3
